# Agent Instructions for docs/

- Todas as páginas `.md` deste diretório e subpastas devem começar com um título na primeira linha (ex.: `# Título`), seguido de uma linha em branco antes do restante do conteúdo.
- Prefira nomes de arquivos em CAIXA ALTA com hífens (`EXEMPLO-GUIA.md`) ao adicionar ou renomear documentos.
- Evite duplicar instruções entre guias; consolide conteúdo redundante ao atualizar docs.
